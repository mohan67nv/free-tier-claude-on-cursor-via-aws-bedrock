## 💡 What Is This?

This is a collection of **step-by-step, beginner-friendly guides** that show developers how to access frontier AI models — Claude, Gemini, and more — at **$0 out-of-pocket** by using the free credits every major cloud provider gives to new accounts.

No subscriptions. No startup required. No enterprise plan. Just your personal cloud account and ~30 minutes.

## 🗺️ Can I Use Both?

Yes. Run both setups simultaneously:
- **AWS Bedrock** connects directly inside Cursor's built-in model settings
- **GCP Vertex AI** runs through a local LiteLLM proxy on a different port

You will have access to Claude (via AWS), Gemini (via GCP), and Claude (via GCP) — all at the same time, all from the same Cursor IDE.
